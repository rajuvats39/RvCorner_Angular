Using Angular Router (Passing Data via Route Parameters)

You can pass data between components using Angular Router. This is especially useful when navigating between components and passing data through URLs or query parameters.

Route Parameters: Use when the data is part of the route path (e.g., /details/123).

Query Parameters: Use when you have optional or additional data to pass (e.g., /details?id=123&name=Raju).

State Object: Use when you don't want to expose the data in the URL.
