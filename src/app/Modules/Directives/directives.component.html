Directives

01) Directives are simply an instruction to the DOM.

02) Directives are simply an TypeScript class.

Types of Directives

Component Directives
01 Component/Custom Directives 02 Structral Directives 03 Attribute Directives

01) Component directives are based on component. Actually, each component can be used as directive. Component provides @Input and @Output decorator to send and receive information between parent and child components.

1) @Input.

2) @Output.

Structral Directives

01) Structural directives change the structure of DOM by adding or removing elements.

1) *ngIf directive.

2) *ngFor directive.

3) *ngSwitch.

Attribute Directives

01) Attribute Directives are used like an attribute on an existing webpage element to change its look and behaviour.

1) ngStyle.

2) ngClass.

3) ngClass.
