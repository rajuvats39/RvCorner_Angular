*ngIf Directives

01) The ngIf is a structural directive.

02) The ngIf is used to add or remove the element from a webpage based on a given condition.

03) If the condition assigned to ngIf returns 'true' it will add the element on which it is used to the webpage, otherwise if the condition returns false then it will remove that element from the webpage.

Eg.1
ngIf

ShowMe is checked

ShowMe is unchecked

ngIf Else

ShowMe is checked

ShowMe is unchecked Using elseBlock
ngIf then else

This is not rendered

ShowMe is unchecked Using elseBlock
