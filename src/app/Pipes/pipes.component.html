Pipes

01)Pipes are a way to transform data in the template directly. They allow you to format, modify, or manipulate values such as strings, dates, numbers, and more without changing the component class itself. Pipes can be built-in or custom.

02) The Angular pipes are used to Transform the Data. For Example, the Date pipe formats the date according to locale rules. We can pass arguments to pipe and chain pipes. Angular also allows us to create the Custom Pipe.

03) Angular Pipes takes data as input and formats or transform the data to display in the template.

Built-in Pipes

Angular provides several built-in pipes to handle common transformations, such as formatting dates, numbers, and currencies, and transforming strings (e.g., uppercase, lowercase). Here are some of the most commonly used built-in pipes:

Data Pipe

medium : {{toDate | date:'medium'}}

short : {{toDate | date:'short'}}

fullDate : {{toDate | date:'fullDate'}}

longDate : {{toDate | date:'longDate'}}

mediumDate : {{toDate | date:'mediumDate'}}

shortDate : {{toDate | date:'shortDate'}}

mediumTime : {{toDate | date:'mediumTime'}}

dd-MM-y : {{toDate | date:'dd-MM-y'}}

dd-MM-yy HH:mm : {{toDate | date:'dd-MM-yy HH:mm'}}

Uppercase & Lowercase Pipe

Unformatted :{{msg}}

Uppercase :{{msg | uppercase}}

Lowercase :{{msg | lowercase}}

Slice Pipe

Complete String :{{msg}}

Example 1 :{{msg | slice:11:20}}

Example 2 :{{msg | slice:-9}}

Decimal / Number Pipe

Unformatted :{{num}}

Formatted :{{num | number}}

Formatted :{{num | number:'3.1-2'}}

Formatted :{{num | number:'7.1-5'}}

Percente Pipe

Unformatted :{{per}}

Example 1 :{{per | percent }}

Example 2 :{{per | percent:'1.2-2'}}

Currency Pipe

Unformatted :{{cur}}

Example 1 :{{cur | currency }}

Example 2 :{{cur | currency:'INR':true:'4.2-2'}}

Json Pipe

{{ {name: 'John', age: 30} | json }}

Async Pipe

{{ data$ | async }}
